# AntdFront

AntdFront 是一套基于 Ant Design 的实验性、拥抱 react hook,纯函数,实现按钮级权限，使用下一代数据流, 微前端框架 qiankun 的 多标签 React 前端模板

## 特性

- **原生:** 纯 hook，同时采用 webpack、babel、antd 等较为原生的配置和函数组件搭建，支持 pwa
- **下一代数据流:** 集成下一代 React 状态管理器
- **微前端:** 使用 qiankun 实现微前端架构
